🤖 四层 Agent 架构

① 数据采集 Agent
生成 120 名员工 mock 数据（HR/OKR/360评估/业务指标），定时刷新，模拟多系统数据汇聚
↓ 结构化员工绩效数据
② 分析 Agent
调用 MiniMax M2.7，对每位员工进行得分分析、履职评估，输出升职/调薪/淘汰/1:1 建议
↓ 结构化建议列表
③ 辅助决策 Agent（Nerve Center）
汇总分析结果，生成日/周/月报告，推送给总经理 Dashboard，支持一键确认或驳回
↓ 总经理决议
④ 执行 Agent
接收决议后自动执行：更新 HR 系统职级、触发调薪流程、发送 1:1 邀请、记录月会决议
