绩效管理 Agent 系统 — 整体架构方案

请选择你最认可的整体技术架构方向

A

全栈 Web 应用（推荐）

前端：Next.js 15 + React 19 + Tailwind CSS v4
后端：Python FastAPI + Claude API（多 Agent 编排）
特点：完整的 Dashboard UI、实时推送、Agent 可视化执行流程
适合：演示给管理层，交互体验最佳

B

纯 Python 后端 + 简单前端

前端：Streamlit 或 Gradio（快速搭建）
后端：Python + Claude API
特点：开发速度最快，2-3天可出 demo
适合：内部技术验证，不追求 UI 精美度

C

纯前端 Mock Demo

前端：Next.js 15 + 全部 mock 数据
后端：无（或极简 mock API）
特点：最快出视觉效果，无需配置 AI 环境
适合：PPT 级别演示，展示 UI 交互流程
